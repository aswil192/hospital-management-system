COMMON WORKFLOWS:
================

1. PATIENT BOOKING APPOINTMENT:
   - Login as patient
   - Go to "Book Appointment"
   - Select specialization (optional filter)
   - Choose doctor
   - Select date and time
   - Enter symptoms
   - Submit booking

2. DOCTOR MANAGING APPOINTMENTS:
   - Login as doctor
   - View pending appointments on dashboard
   - Go to "My Appointments"
   - Confirm pending appointments
   - Complete confirmed appointments
   - Create bills for completed appointments
   - Prescribe medicines if needed

3. ADMIN ADDING NEW DOCTOR:
   - Login as admin
   - Go to "Manage Doctors"
   - Click "Add New Doctor"
   - Fill in all required information
   - Set salary amount
   - Submit

4. ADMIN MANAGING MEDICINE INVENTORY:
   - Login as admin
   - Go to "Manage Medicines"
   - Click "Add New Medicine" to add new
   - Click "Stock" button to update quantity
   - Low stock items highlighted in yellow

5. PATIENT PAYING BILL:
   - Login as patient
   - Go to "My Bills"
   - Click "Pay Now" on unpaid bill
   - Select payment method
   - Enter amount
   - Confirm payment

6. ADMIN PAYING DOCTOR SALARY:
   - Login as admin
   - Go to "Doctor Salaries"
   - Click "Pay Salary"
   - Select doctor
   - Amount auto-fills with monthly salary
   - Select payment month
   - Add notes (optional)
   - Process payment

IMPORTANT NOTES:
===============

SECURITY:
- Never share admin credentials
- Change default passwords in production
- All passwords are hashed in database
- Sessions expire on browser close

DATABASE:
- Database name: hospital_management
- All timestamps in UTC
- Cascading deletes enabled for related records

APPOINTMENT STATUSES:
- pending: Newly booked, awaiting confirmation
- confirmed: Doctor confirmed the appointment
- completed: Appointment finished
- cancelled: Patient cancelled

BILL STATUSES:
- unpaid: No payment received
- partially_paid: Some payment received
- paid: Full payment received

DOCTOR AVAILABILITY:
- available: Accepting appointments
- unavailable: Not accepting appointments
- on_leave: Temporarily unavailable

QUICK TIPS:
==========

1. To test the complete flow:
   - Register as new patient
   - Book appointment with doctor
   - Login as doctor and confirm appointment
   - Complete appointment and create bill
   - Login as patient and pay bill

2. All email addresses must be unique

3. Doctors can only create bills for completed appointments

4. Patients can only book with available doctors

5. Admin can view all system data but cannot book appointments

6. Flash messages auto-hide after 5 seconds

7. Tables are responsive and work on mobile devices

8. All dates display in readable format (e.g., "Nov 01, 2025")

9. All currency displays in USD format (e.g., "$100.00")
